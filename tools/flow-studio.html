PRISMA flow

The PRISMA mode follows the PRISMA 2020 flow diagram for new systematic reviews: a databases-and-registers arm and a separate other-methods arm, both feeding the included box, with an optional previous-version column for updated reviews. The layout and wording follow Page MJ et al., The PRISMA 2020 statement (BMJ 2021;372:n71). The numbers are yours to enter — the tool draws the boxes and arrows, it does not check that the arithmetic balances. A box line is shown only when you give it a value, so leave a field blank to drop that line.

Funnel

The funnel mode takes an ordered list of stages, each with a count, and draws a tapering flow. Each stage is sized to its count; the share of the first stage and the drop-off to the next stage are calculated automatically. It suits application and grant pipelines, recruitment, or any sequential review where you want to show attrition.

Output

Both produce a single self-contained SVG with an explicit viewBox, set in DM Sans to match the other tools. Copy the SVG for the web or further editing, or export a PNG for Office. Diagrams size themselves to their content.

Caveats

This is a layout tool, not a reporting checklist. For a published review, follow the full PRISMA 2020 reporting guidance and check every count against your records. The sample figures are illustrative — replace them before sharing.
